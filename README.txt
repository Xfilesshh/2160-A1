NthWord Directions:
To compile the Nthword file, type "Clang -Wall nthwords.c -o NthWord" into the Terminal.
To run NthWord, type "NthWord < nthword.txt" into the terminal.

Levenshtein Directions:
To compile the Levenshtein file, type "Clang -Wall levenshtein.c -o Levenshtein" into the Terminal.
To run Levenshtein, type "./Levenshtein" into the terminal.
